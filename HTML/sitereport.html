Coding Experience

I was familiar with HTML and CSS as I had prior experience using it. But, this project was not about using HTML and CSS, it was about using it properly and efficiently to come up with a professional looking and appealing portfolio website.

For my portfolio website, I used HTML and CSS to showcase my skills in web designing. Before starting actual coding, I sketched some ideas for the website's design and layout. I checked out some examples in the internet and took inspiration for the design of the website. Talking about the design of the website, I tried to keep it simple, clean and compact. I avoided making it colorful and using various background images. The color theme of the website is black. I choose black as the primary color because I think it gives a sophisticated look to the website. And other colors I used are contrasting color that goes well with black and complements it.

As for my HTML, I used <header>, <main> and <footer> tags to divide the website into three main parts. Inside these tag, I used <section> and <div> tags to manage the content inside each part of the website.
